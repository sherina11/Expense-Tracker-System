# 💰 Expense Tracker System (C Project)

## 📌 Description
This project is developed using the C programming language.  
It is a simple Expense Tracker System used to store and calculate daily expenses.

---

## 🎯 Features
- Add daily expenses
- View all expenses
- Calculate total spending
- Colored console interface
- Menu-driven design

---

## 🛠️ Technologies Used
- C Programming Language
- GCC Compiler
- VS Code / CodeBlocks

---

## ▶️ How to Run

### Compile
gcc expense_tracker.c -o expense

### Run
./expense

(For Windows)
expense.exe
